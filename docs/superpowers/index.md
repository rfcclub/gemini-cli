# Vesta's Superpowers

Vesta's Superpowers introduce premium capabilities that elevate Gemini CLI into
an advanced, highly efficient, and visually rich AI-native engineering partner.
These features maximize the unique strengths of Gemini models, specifically the
massive context window, native multimodal vision, and high-performance context
caching.

By integrating these features, you experience significantly reduced operational
costs, pixel-perfect user interface generation, highly resilient file edits, and
real-time diagnostics of terminal sessions.

---

## Core Superpowers

Every superpower focuses on solving a distinct operational friction point in AI-
guided development workflows.

### Context Caching

Automated prefix caching cuts API token consumption costs and speeds up model
response times during long active terminal sessions. Refer to the
[Context Caching](./context-caching.md) guide for details.

### Resilient Block Replace

A formatting-insensitive search and replacement algorithm stops formatting
mismatches (such as mixed indents or line endings) from breaking file edits.
Refer to the [Resilient Block Replace](./resilient-replace.md) guide for
details.

### Visual UI Forge

A native multimodal tool lets you submit high-fidelity image mockups alongside
frontend source files to automatically align card designs, paddings, and
gradients. Refer to the [Visual UI Forge](./visual-ui-forge.md) guide for
details.

### Analytics Dashboard

A real-time, interactive terminal dashboard renders token metrics, cache hits,
and USD savings directly inside the terminal interface. Refer to the
[Analytics Dashboard](./analytics-dashboard.md) guide for details.
